# 美术资源使用说明

本文件夹存放所有美术资源，包括角色立绘、背景图、UI素材等。

## 文件夹结构

```
resources/
├── characters/      # 角色立绘
│   ├── xuena/      # 雪乃（妈系学姐）
│   ├── lingyin/    # 玲音（萝莉系学妹）
│   ├── xiaoshuang/ # 晓霜（傲娇系同班）
│   ├── yeyu/       # 夜雨（神秘系转学生）
│   ├── xinghe/     # 星河/夏奈（元气系好友）
│   └── README.md   # 角色立绘说明
├── backgrounds/    # 背景图
├── achievements/    # 成就图标
├── ui/             # UI素材
└── scripts/        # 对话脚本
```

## 在Godot中使用角色立绘

### 方法1：在代码中使用

```gdscript
# 加载角色立绘
var texture = load("res://resources/characters/xuena/xuena_default.png")

# 在Sprite2D中显示
$Sprite2D.texture = texture

# 切换表情
func show_expression(character: String, expression: String):
    var path = "res://resources/characters/%s/%s_%s.png" % [character, character, expression]
    $Sprite2D.texture = load(path)
```

### 方法2：在对话系统中使用

在对话脚本中，可以添加 `@show` 指令来显示角色：

```
@show xuena default left
旁白: 雪乃走了过来...
雪乃: 早上好！
```

需要在DialogManager中实现 `@show` 指令的支持。

## 在Godot中使用背景图

```gdscript
# 加载背景图
var bg_texture = load("res://resources/backgrounds/bg_1.png")

# 在背景Sprite中显示
$BG/Sprite2D.texture = bg_texture

# 切换背景
func change_background(bg_index: int):
    var path = "res://resources/backgrounds/bg_%d.png" % bg_index
    $BG/Sprite2D.texture = load(path)
```

## 测试角色显示

项目包含一个角色显示测试脚本 `scripts/character_display.gd`。

使用方法：
1. 在Godot中创建新场景
2. 添加Node2D作为根节点
3. 添加Sprite2D子节点（显示角色立绘）
4. 添加CanvasLayer子节点（显示UI）
5. 将 `character_display.gd` 脚本挂载到根节点
6. 运行场景，使用按键测试：
   - 按键1：切换角色
   - 按键2：切换表情
   - 按键3：左侧显示
   - 按键4：右侧显示
   - ESC：退出
